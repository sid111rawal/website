## Recommended Image Specifications:
- **Format**: JPG or PNG
- **Dimensions**: 1200x800px (3:2 aspect ratio) or 1600x900px (16:9)
- **File Size**: Under 500KB per image (optimize for web)
- **Quality**: High quality but optimized for fast loading

## How to Get Website Screenshots:
1. Use browser extensions like "Full Page Screen Capture"
2. Use online tools like:
   - https://screenshot.guru/
   - https://www.screenshot.rocks/
   - https://htmlcsstoimage.com/
3. Take manual screenshots and crop/edit as needed

## How to Get Instagram Screenshots:
1. Take screenshots of Instagram profile/grid
2. Create a collage of top performing posts
3. Use Instagram's native screenshot feature
4. Use tools like Canva to create professional showcase images
